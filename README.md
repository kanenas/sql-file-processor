# SQL file processor

A comprehensive web version of the **SQL file processor**

Features:
- Modern Web Interface: Responsive design with progress indicators
- Large File Support: Handles files up to 500MB
- Real-time Progress: Visual feedback during processing
- Table Selection: Choose specific tables to split
- Secure Downloads: Protected file download system
- Error Handling: Comprehensive error messages
- Mobile Friendly: Works on all device sizes
